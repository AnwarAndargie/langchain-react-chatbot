# Frontend

React + TypeScript + Vite UI for the chatbot. Auth (login/signup), chat with sidebar and conversation list.

## Run

```bash
npm install
npm run dev
```

Set `VITE_API_URL` in `.env` (default `http://localhost:8000`).

## Scripts

- `npm run dev` — dev server
- `npm run build` — production build
- `npm run preview` — preview build
